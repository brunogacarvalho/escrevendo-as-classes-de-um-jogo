# Desafio: Classes de um Jogo

Projeto desenvolvido como desafio da DIO.

## Descrição

Este projeto cria uma classe Heroi que representa um personagem de aventura. Cada herói possui nome, idade e tipo, e pode realizar ataques diferentes dependendo do seu tipo.

## Tipos de Heróis e Ataques

- Mago: ataca usando magia
- Guerreiro: ataca usando espada
- Monge: ataca usando artes marciais
- Ninja: ataca usando shuriken

## Como funciona

O código cria um herói e executa o método atacar(), que exibe uma mensagem personalizada de acordo com o tipo do herói.
